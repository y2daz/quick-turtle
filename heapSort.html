Heap Sort Algorithm Simulator

This simulator models the behaviour of the heap sort algorithm. For simplicity, only 2 digit numbers are allowed to be sorted (including the - sign on negative numbers.

The Algorithm

Procedure HeapSort( A )
1 BuildHeap ( A )
2 for i ← A.length down to 2
3 exchange A[ 1 ] ↔ A[ i ]
3 A.heapSize ← A.heapSize - 1
3 Heapify( A, 1)

Procedure BuildHeap( A )
1 A.heapSize ← A.length
2 for i ← ⌊ A.heapSize / 2 ⌋ down to 1
3 Heapify ( A , i )

Procedure Heapify( A, i )
1 l ← leftChild( i )
1 r ← rightChild( i )
2 if l ≤ A.heapSize and A[ l ] > A[ i ]
3 largest ← l
2 else
3 largest ← i
2 if r ≤ A.heapSize and A[ r ] > A[ largest ]
3 largest ← r
2 if largest ≠ i
3 exchange A[ i ] ↔ A[ largest ]
3 Heapify( A, largest )
