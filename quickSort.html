Quick Sort Algorithm Simulator

This simulator models the behaviour of the quick sort algorithm. For simplicity, only 2 digit numbers are allowed to be sorted (including the - sign on negative numbers.

The Algorithm

Procedure QuickSort( A, p, r )
1 if ( p < r ) then
2 q ← Partition( A, p, r )
3 QuickSort( A, p, q - 1 )
4 QuickSort( A, q + 1, r )

Procedure Partition( A, p, r )
1 x ← r
2 i ← p
3 for ( j ← p; j ≤ ( r - 1 ); j++ )
4 if ( A[j] < A[x] )
5 exchange A[i] ↔ A[j]
6 i ← i + 1
7 exchange A[i] ↔ A[r]
8 return ( i )
